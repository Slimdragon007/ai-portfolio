## My AI Philosophy

**Zero-based thinking**: every component in a system needs a specific job. If it doesn't earn its place, it gets cut.

**4D Framework**: Define (why + goal) -> Delegate (what goes where) -> Discern (key decisions) -> Diligence (execute and verify)

**Agent orchestration over chatbot prompting**: the real unlock isn't a single prompt. It's context routing, persistent state, and tool integration working as one layer.
